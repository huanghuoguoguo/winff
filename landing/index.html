快速部署

只需简单几步，即可在您的 Windows 或 Linux 服务器上运行。

  • 1 检出代码库并安装依赖
  • 2 配置共享目录 config.json
  • 3 运行 start.bat
config.json 
{
  "port": 3000,
  "sharedDirs": {
    "视频库": "D:\\Videos",
    "项目工程": "E:\\Projects"
  }
}
